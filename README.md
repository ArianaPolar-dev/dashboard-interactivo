# Dashboard Interactivo

## Descripción

Dashboard avanzado para gestión de tareas, seguimiento de hábitos, panel administrativo y visualización de datos. El proyecto está desarrollado con React, Vite y Tailwind CSS, aplicando buenas prácticas de arquitectura, hooks personalizados, diseño responsive y visualizaciones con Recharts.

## Características principales

- Múltiples vistas: Dashboard, Gestor de tareas, Tracker de hábitos, Panel de administración y Analytics.
- Estado global eficiente mediante hooks personalizados.
- Componentes reutilizables y arquitectura modular.
- Visualizaciones interactivas con Recharts.
- Diseño adaptativo para diferentes dispositivos.
- Animaciones sutiles para mejorar la experiencia de usuario.

## Tecnologías utilizadas

- React 18
- Vite
- Tailwind CSS
- Recharts
- Lucide Icons
